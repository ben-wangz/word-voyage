# Sample Data for OpenAI LLM E2E Tests

This directory contains sample data used by the end-to-end tests for the OpenAI LLM Service.

## Usage

The samples are embedded in the test code and used for:

- Testing basic structured generation
- Validating context size limits
- Testing schema validation
- Streaming generation tests

## Game Context Examples

The samples include typical game state contexts for a space exploration survival game:

- Player health, hunger, thirst levels
- Current location information
- Energy and resource tracking
- Event generation prompts and user inputs
